# Color Palette

Based on the darker, more muted theme from the screenshot.

## Backgrounds

| Name | Hex | Usage |
|------|-----|-------|
| Base | `#15152a` | Main page background, input fields, tooltip backgrounds |
| Surface | `#191831` | Cards, elevated surfaces, chart area backgrounds, favicon |
| Elevated | `#252133` | Interactive elements, hover states |
| Overlay | `#2D2640` | Hover states, overlays |

## Borders

| Name | Hex | Usage |
|------|-----|-------|
| Subtle | `#3D3554` | Card borders, dividers, chart grid lines, axis domain lines |
| Muted | `#4A4166` | Input borders, focus states |

## Text

| Name | Hex | Usage |
|------|-----|-------|
| Primary | `#E8E2F4` | Headings, titles |
| Secondary | `#a196e4` | Labels, body text, chart axis legends, tooltip text |
| Muted | `#6B6483` | Placeholders, disabled text, chart legend items |
| Input | `#fad003` | Input field values |
